# EventAppP1
Aplikacja do tworzenia i obsługi wydarzeń

## server
Przed uruchomieniem:
```
cd .\server\
```
```
npm install
```

Następnie uruchomić poleceniem:
```
npm start
```
Działa na porcie 5000

## client
Przed uruchomieniem:
```
cd .\client\
```
```
npm install
```

Następnie uruchomić poleceniem:
```
npm start
```
Działa na porcie 3000
